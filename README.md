# Greddiit Website

## Website Features-
 - Every user has a unique email
 - Password is hashed
 - On Login you go to empty dahsboard page from where you can redirect to Profile, subGreddits etc
 - Number of people in subgreddiit is number who have joined
 - You must enter tags and banned keywords as comma separated values
 - Join requests disappear upon accepting/rejecting
 - User can only view the joined subgreddiit
 - A post can have multiple comments, the comments are anonymous
